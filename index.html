다크 모드(Dark Mode) 지원

GFM 콜아웃(Alert Box) 가시성 보완

다단계 제목 및 목록 구성 기능 추가

  • 들여쓰기/내어쓰기 버튼 및 Tab 단축키 지원

사용자 지정 마크다운 PWA 앱 아이콘 탑재

반응형 전체 너비(Wide View) 레이아웃 적용

  • 에디터 단독 보기 가로폭 대폭 확장

제목 스타일 선택 메뉴 잘림(Clipping) 버그 수정

  • 드롭다운 메뉴를 body 하위 최상위 레이어로 구조 분리
  • fixed 절대 좌표 매핑으로 스크롤 잘림 문제 해결
v1.7.0 2026-06-18

실행 취소(Undo) 및 실행 복귀(Redo) 기능 구현

  • 에디터 내 텍스트 편집 및 마우스 동작 상태 스택 관리
  • 단축키 연동 (Ctrl+Z / Ctrl+Y / Cmd+Shift+Z 등)
  • 도구 모음 우측 끝에 전용 실행 취소/복귀 버튼 배치
v1.8.0 2026-06-23

에디터 기본값 복원(초기화) 및 안전장치 강화

  • 에디터 상단에 디폴트 템플릿 복원을 위한 '초기화' 버튼 신설
  • 새 문서 시작, 파일 로드, 초기화 시 실수 방지를 위한 Undo(실행 취소) 연동
v1.9.0 2026-06-23

이모지(Emoji) 리스트 삽입 기능 추가

  • 툴바에 특수 장식 블록으로 '이모지' 버튼 추가
  • 그리드 형태의 fixed 팝오버를 띄워 30여종의 핵심 이모지 즉시 선택 및 삽입
  • 이모지 선택 입력 시 히스토리 스택 연동으로 즉각 실행 취소(Undo) 가능
v2.0.0 2026-06-23

이모지 픽커 대폭 개편 및 150종 이상 제공

  • 150종 이상의 다채로운 이모지 셋(표정, 동물, 음식, 여행, 물건 등) 구축
  • 이모지 검색 기능(실시간 한글/영어 키워드 매칭) 도입
  • 카테고리별 탭 전환식 픽커 UI 구현
v2.1.0 2026-06-23

표현 가능한 대규모 이모지 세트 탑재

  • 약 600여 종에 달하는 풍부한 표준 상용 이모지 목록 추가
  • 카테고리 탭 구성을 최적화하여 방대한 데이터의 스크롤 탐색 성능 개선
v2.1.1 2026-06-23

이모지 픽커 탭 전환 버그 수정

  • 다른 이모지 탭으로 이동 시 메뉴가 의도치 않게 닫히는 현상(Detached Node 오류) 해결
  • 팝업 및 드롭다운의 외부 클릭(Click Outside) 이벤트 안전성 강화
v2.2.0 2026-06-23

동적 표 생성기 추가

  • 표 아이콘 및 슬래시 메뉴(/표) 사용 시 가로/세로 지정 가능한 팝업창 추가
  • 입력한 크기 및 조건에 맞춰 마크다운 테이블 자동 삽입되도록 기능 개선
v2.3.0 2026-06-23

다중 포맷 저장(Export) 지원

  • 마크다운(.md) 외에 텍스트(.txt), Word(.docx), PDF(.pdf) 확장자 저장 기능 추가
  • 직관적인 저장 옵션 선택 드롭다운 UI 적용
  • Word/PDF 저장 시 우측 미리보기 레이아웃(문법이 적용된 결과물) 유지 기능 탑재
