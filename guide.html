Technique 01

제목 (Headers)

문법 (Syntax)

# H1 가장 큰 제목
## H2 두 번째로 큰 제목
### H3 세 번째로 큰 제목
#### H4 네 번째로 큰 제목
##### H5 다섯 번째로 큰 제목
###### H6 가장 작은 제목

결과 (Result)

H1 가장 큰 제목

H2 두 번째로 큰 제목

H3 세 번째로 큰 제목

H4 네 번째로 큰 제목

Technique 02

텍스트 강조 (Text Styling)

문법 (Syntax)

**굵게 (Bold)**
*기울임 (Italic)*
***굵고 기울게***
~~취소선 (Strikethrough)~~

결과 (Result)

굵게 (Bold)

기울임 (Italic)

굵고 기울게

취소선 (Strikethrough)

Technique 03

목록 (Lists)

문법 (Syntax)

- 순서가 없는 목록 1
- 순서가 없는 목록 2
- 하위 목록

1. 순서가 있는 목록 1
2. 순서가 있는 목록 2

- [ ] 미완료 할일
- [x] 완료된 할일

결과 (Result)

  • 순서가 없는 목록 1
  • 순서가 없는 목록 2
    • 하위 목록
  1. 순서가 있는 목록 1
  2. 순서가 있는 목록 2
미완료 할일
완료된 할일
Technique 04

링크 및 이미지 (Links & Images)

문법 (Syntax)

[구글 링크](https://google.com)

![대체 텍스트](image_url)

[![클릭가능한 이미지](image_url)](https://google.com)

결과 (Result)

구글 링크
이미지 표시 영역
Technique 05

코드 (Code)

문법 (Syntax)

`인라인 코드`를 입력할 때 사용합니다.

```python
def hello_world():
    print("Hello, Markdown!")
```

결과 (Result)

인라인 코드를 입력할 때 사용합니다.

def hello_world():
    print("Hello, Markdown!")
Technique 06

표 (Tables)

문법 (Syntax)

| 이름 | 나이 | 직업 |
| :--- | :---: | ---: |
| 철수 | 25 | 개발자 |
| 영희 | 28 | 디자이너 |

* 콜론(:)으로 정렬(좌,중,우) 설정 가능

결과 (Result)

이름 나이 직업
철수 25 개발자
영희 28 디자이너
Technique 07

인용구 (Blockquotes)

문법 (Syntax)

> 첫 번째 인용구입니다.
>> 중첩된 인용구도 가능합니다.
> 인용문 안에 **굵은 글씨**도 됩니다.

결과 (Result)

첫 번째 인용구입니다.
중첩된 인용구도 가능합니다.

인용문 안에 굵은 글씨도 됩니다.
